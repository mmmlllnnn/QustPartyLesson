## 我已经上传到Greasy Fork平台

## 使用方法：
1. 浏览器安装Tampermonkey(油猴)插件。
2. 点击上面提到的Greasy Fork平台，点击脚本安装按钮。
3. 进入青科党课学习网站，进入理论学习后，手动点击一章进入，会自动开始播放并往下跳转。


可以秒刷，代码开头标注了延迟时间，如果网络差可以自行改高。
